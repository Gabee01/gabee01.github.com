ExChange - Currency Conversion API

A robust API for converting amounts between various currencies

Project Description

ExChange is a powerful and flexible currency conversion API built with Elixir and Phoenix. It provides real-time currency conversion rates and allows users to convert amounts between different currencies efficiently.

Technologies Used

  • Elixir
  • Phoenix Framework
  • ExchangeRate API integration
  • Docker
  • OpenAPI 3.0 for documentation

Key Features

  • Real-time currency conversion
  • Support for multiple currencies
  • RESTful API design
  • Microservice architecture
  • Comprehensive API documentation
  • Dockerized for easy deployment

Challenges and Solutions

Challenge: Ensuring up-to-date exchange rates

Solution: Implemented a caching mechanism with regular updates from the ExchangeRate API to balance between having current rates and minimizing API calls.

Challenge: Handling high concurrency

Solution: Leveraged Elixir's concurrency model and OTP to create a highly scalable system capable of handling multiple requests simultaneously.

Challenge: Ensuring API reliability

Solution: Implemented comprehensive error handling and fallback mechanisms to handle potential issues with the external exchange rate service.
